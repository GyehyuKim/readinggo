## 1. 한 줄 정의

> *"책을 책장에서 꺼내게 만든다."*  
> 듀오링고 게이미피케이션(스트릭·XP·소셜)을 독서 루틴에 이식한 습관 형성 앱

---

## 2. 화면 구조 (IA)

```
앱
├── 상단바 (고정)
│   ├── readingGo 로고
│   ├── 🔥 스트릭 · ⚡ XP · Lv 배지 · 🪶 방패 수
│   └── 🔍 도서 검색 아이콘 → [검색 모달]
│
├── 탭 1 — 둥지 (홈)
├── 탭 2 — 마을
├── 탭 3 — 소셜
└── 탭 4 — 프로필
```

> **내서재 탭은 프로필로 변경.**

## 3. 탭별 상세 스펙

---

### 3-1. 둥지 탭 (홈)

화면 스크롤 순서:

#### ① 현재 읽는 책
- 둥지에 노출되는 책은 **1권** (현재 활성 책)
- 책 표지·제목·저자·출판사·현재 페이지 / 전체 페이지 표시 (진행률)
- **[책 변경]** 버튼: 탭 시 내가 읽고 있는 책 목록 우선 노출 → 선택 시 즉시 전환. 검색으로 새 책 추가도 가능
- 책을 변경해도 이전 책은 프로필 "읽고 있는 책"에 진행률과 함께 유지됨 (중단 표시 없음)
- 현재 책 없으면 "지금 읽는 책을 등록해보세요" CTA

#### ② 내 둥지 상태
- 둥지 레빌(Lv)·둥지 시각화·체력 게이지(XP)·설명 문구 표시
- 둥지 시각화 (XP 획득 시 둥지 단계별 확장, XP 차감 시 둥지 단계별 소실)  

#### ③ 오늘의 한쪽 (데일리 기록 입력)
| 입력 항목 | 설명 |
|---|---|
| 오늘 읽은 마지막 페이지 | 필수. 숫자 입력 |
| 기억하고 싶은 한 문장 | 필수. 최대 200자 |
| 스포일러 포함 여부 | 체크박스. 체크 시 다른 유저에게 블러 처리 |

- 등록 완료 시 즉시 보상: XP 획득 + 스트릭 유지
- 오늘 이미 기록했어도 수정 및 복수 등록 가능
- 미기록 시: 🪶 방패 보유 시 → 방패 1개 소모, 스트릭 유지 / 방패 없을 시 → 스트릭 초기화 + XP -5
- 완독 시 완독 세레머니 + 200XP 보상 획득 + 소감 및 책장 등록

#### ④ 내 한문장
- 내가 기록한 한 문장 목록 (최신순)
- **캘린더 ↔ 리스트 뷰 전환 토글**
  - 캘린더: 월간 캘린더, 기록한 날짜 초록 점 표시, 날짜 탭 시 해당일 문장 팝업
  - 리스트: 날짜 + 페이지 + 문장 카드 + 좋아요 표시
- 스포일러 체크된 문장은 본인에게는 "⚠️ 스포일러" 태그 표시

### 3-2. 마을 탭

#### 마을 목록 화면
- 내가 참여 중인 마을 카드 목록
  - 책 표지 · 마을명 · 참여 인원 · 현재 파트 · D-day
- **[+ 마을 만들기]** 버튼

#### 마을 개설 플로우
| 단계 | 입력 항목 |
|---|---|
| 책 선택 | 검색 모달에서 1권 선택 |
| 마을 정보 | 마을 이름 · 소개 (선택) |
| 공개 설정 | 공개(검색 노출) / 비공개(친구검색·링크·코드 초대) |
| 마일스톤 설정 | 책을 N개 파트로 분할 → 각 파트 마감일 지정 |
| 멤버 초대 | 링크·코드 공유 또는 유저 검색으로 초대 |

#### 마을 참여 방식
- **공개 마을**: 도서 상세 페이지 → "이 책 읽는 마을" 섹션 → [참여] 버튼
- **비공개 마을**: 초대 푸쉬 알림·코드 입력·카카오톡 등 링크 수신 → 1탭 입장

#### 마을 내부 화면 (스크롤 순서)
1. **마일스톤 진행 바**: 현재 파트 / 전체 파트 · 마감 D-day
2. **참여자 둥지 그리드**: 집 아이콘 모아보기
   - 불빛 ON = 오늘 기록 완료
   - 불빛 OFF = 미기록 → [모이 보내기] 버튼
3. **참여자 탭 시 프로필 팝업**
   - 오늘 기록 여부 · 오늘 한 문장 · 누적 페이지 수
   - 프로필 바로가기 (책장 · 스트릭 · XP)
4. **파트별 랭킹**: 마을 지정 책에 대한 기록만 카운트한 누적 페이지 수 기준 순위. 다른 책 기록은 포함되지 않음
5. **여러 마을 동시 참여 가능**: 둥지 탭 기록이 마을 진행률에 자동 반영

### 3-3. 소셜 탭

#### 신규 유저 (팔로우 0명)
- "같은 책 읽는 사람 찾기" / "친구 찾기" / "마을 참여하기" CTA 온보딩 화면

#### 팔로우 피드
- 팔로우한 사람들의 한 문장 카드 (최신순)
- 카드 구성:  닉네임 · 책 제목 · 책 페이지 · 문장 · 작성시간 · 박수 수 · 책갈피 수
- 스포일러 체크된 문장: "⚠️ 스포일러 포함 — 탭하면 보기" 블러 처리
- 👏 박수 버튼: 탭 시 +1 (내 프로필에서 확인 가능)
- 책갈피 버튼 : 탭 시 +1 (내 프로필에서 확인 가능)
- 프로필 탭 시 → 해당 유저 프로필 이동
- 문장 카드 탭 시 → 도서 상세 페이지로 이동 

#### 전체 리그 (주간)
- 앱 전체 주간 XP 순위 상위 N명
- 내 현재 순위 · 바로 위/아래 유저 비교 표시

### 3-4. 프로필 탭

#### 내 프로필 상단
| 요소 | 내용 |
|---|---|
| 닉네임 · 한 줄 소개 | 편집 가능 |
| 팔로워 / 팔로잉 수 | 탭하면 목록 확인 |
| 문장 좋아요·책갈피 기록 | 탭하면 목록 확인 |
| 둥지 레벨 · 총 완독 권수 · 현재 스트릭 · 총 XP  | 통계 뱃지 |


#### 내 서재 섹션
| 탭 | 내용 |
|---|---|
| 읽고 싶은 책 | 찜한 책 목록. [지금 읽기] 버튼으로 현재 책 전환 |
| 읽고 있는 책 | 읽다가 변경된 책 포함 전체 진행 중인 책 목록 · 각 권 진행률 표시. 탭하면 둥지 활성 책으로 전환 가능 |
| 읽은 책 | 완독한 책 목록 = 책장과 동일 |

#### 책장 (완독 기록)
- **완독한 책만** 책장에 등록 (완독 세리머니 후 자동 저장)
- 완독 시 입력 항목
  - 별점 (1~5, 선택)
  - 완독 소감 텍스트 (선택)
- 책 카드 탭 시
  - 완독 소감 + 별점
  - 해당 책에서 기록한 전체 한 문장 목록 (날짜순/검색 가능)
- **다른 사람 책장 구경 가능** (전체 공개 정책)
  - 스포일러 체크된 문장은 블러 처리

#### 읽고 싶은 책 (찜 목록)
- 도서 검색에서 [❤️ 찜하기] → 프로필 "읽고 싶은 책" 탭에 저장
- 찜한 책은 도서 검색 추천 섹션 상단에도 노출

#### 내 한문장 Export
- **책 단위** 또는 **전체 단위**로 export 가능
- MVP: Markdown (책 제목 + 문장 목록 + 수집 날짜)
- Post-MVP: PDF, 이미지 카드 (인스타 비율, 공유용)
- "이건 다 당신 자산입니다" — 내 서재 첫 진입 시 1회 안내 카드 노출

## 4. 도서 검색 모달 (상단바 🔍 아이콘)

상단바 아이콘 탭 시 하단에서 슬라이드업으로 열림


#### 검색 입력창
- 제목 · 저자 · ISBN 검색
- 검색 결과 카드: 표지 · 제목 · 저자 · 리딩고 등록 수

#### 추천 섹션
1. **친구·마을이 읽는 책**: 팔로우 중인 사람 / 마을 멤버 현재 도서
2. **앱 전체 인기 도서**: 등록 수 상위

#### 도서 상세 페이지
| 섹션 | 내용 |
|---|---|
| 기본 정보 | 표지 · 제목 · 저자 · 전체 페이지 수 |
| 리딩고 현황 | 읽는 중 N명 / 완독 N명 |
| 한 문장 모음 | 이 책 읽은 유저들의 한 문장 (스포 블러 포함) |
| 같이 읽는 마을 | 공개 마을 목록 → [참여] 버튼 |
| 읽은 사람 | 유저 목록 → 프로필 · 팔로우 버튼 |

- [지금 읽기] 버튼: 현재 책으로 지정
- [❤️ 찜하기] 버튼: 읽고 싶은 책으로 저장

| 알림 | 트리거 | 문구 예시 |
|---|---|---|
| 모이 🪱 | 친구가 모이 전송 | "OOO님이 모이를 보냈어요! 오늘 리리를 깨워줘요 🐦" |

## 6. XP 규칙

### 획득 (Gain)

| 이벤트 | XP |
|---|---|
| 데일리 기록 완료 (오늘의 한쪽) | +10 |
| 스트릭 7일 달성 보너스 | +100 |
| 스트릭 30일 달성 보너스 | +500 |
| 완독 | +200 |
| 팔로우 유저에게 박수 받음 | +5 (일 최대 +20) |

### 차감 (Loss)

| 이벤트 | XP |
|---|---|
| 방패 없을 때 미기록 1일 경과 | -5 |
| 최솟값 | 0 (음수 없음) |

> 방패(🪶) 소모로 스트릭 유지 시 XP 차감 없음.

### 방패 (🪶 Shield) 시스템

- **역할**: 1일 미기록 시 방패 1개 소모 → 스트릭 유지, XP 차감 없음
- **기본 충전**: 7일 연속 달성마다 1개 무료 지급
- **상단바**: 스트릭·XP 옆에 🪶 보유 수 표시
- 방패가 없는 상태에서 미기록 → 스트릭 초기화 + XP -5

### 둥지 단계 기준 (현재 누적 XP 구간)

| XP 구간 | 단계 | 아이콘 |
|---|---|---|
| 0 ~ 49 | 나뭇가지 자리 | 🪵 |
| 50 ~ 149 | 빈 둥지 | 🪹 |
| 150 ~ 299 | 따뜻한 둥지 | 🏠 |
| 300 ~ 499 | 다정한 집 | 🏡 |
| 500 ~ | 참새의 성 | 🏰 |

- XP가 오르면 단계 상승, 내려가면 단계 하락 (손실 회피 심리 활용)
- 방패 없이 미기록이 누적되면 집이 낡아지는 시각 연출 적용
- 스트릭과 XP는 독립적으로 관리됨 (스트릭 초기화 ≠ XP 초기화)
- 책을 변경해도 XP·둥지 단계 유지 (책 진행률과 무관)

## 7. 스포일러 처리 규칙

- 한 문장 입력 하단 체크박스: **"스포일러가 포함된 문장이에요"**
- 체크 시 → 피드 · 도서 상세 · 마을 · 책장 모든 곳에서 블러 처리
- 표시 방식: "⚠️ 스포일러 포함 — 탭해서 보기"
- 미체크 → 전체 공개 노출
- 작성자 본인 화면: 블러 없이 "⚠️ 스포일러" 태그만 표시

### 8-2. 회복탄력성 (What-the-hell 방어)

하루 결석이 전체 실패로 규정되면 안 됨.

| 장치 | 동작 |
|---|---|
| 🪶 방패 (Shield) | 자동 적용. 1일 미참여 = 방패 1개 소모, 스트릭 유지. 7일 연속 달성마다 1개 무료 충전 |
| ⏸️ 휴식권 (Pause) | 사용자가 사전에 1~7일 휴식 선언 → 스트릭 동결. 월 1회 무료 |
| 복귀 메시지 | 결석 후 복귀 시 죄책감 유발 X. "다시 시작" 톤만 |

---

### 8-3. 배지

| 달성 조건 | 배지 |
|---|---|
| 7일 연속 스트릭 | 🔥 위클리 스트릭 배지 + XP +100 |
| 30일 연속 스트릭 | 🏆 먼슬리 스트릭 배지 + XP +500 |
| 완독 | 📚 완독 배지 (책 제목 포함) + XP +200 |

### 8-4. 마을 공동 자산

마을 내 참여자 전체가 당일 기록을 완료할 때마다 마을 공동 자산이 성장.

**도서관 짓기** (기본 테마):
- 마을원 전원 당일 기록 완료 = 1 brick 적립
- 1,000 brick = 1 wing 완성
- 시각화: 마을 내부 화면에 건물이 점진적으로 자라남

**세계수 키우기** (대안 테마):
- 1회 전원 완료 = 1 leaf
- 계절별 외형 변화

테마는 마을 개설 시 선택. 이미 쌓인 자산은 특정 멤버의 결석으로 깎이지 않음.

---

### 8-5. "오늘의 문장" Export — 락인 정당화

스트릭 lock-in의 윤리적 부담을 export 약속으로 상쇄.

> *"당신이 모은 문장은 언제든 가져갈 수 있습니다. 그래도 매일 오는 이유는, 우리가 그걸 더 잘 모아주기 때문입니다."*
