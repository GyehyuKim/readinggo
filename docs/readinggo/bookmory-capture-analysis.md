## 2. 핵심 통찰

1. **북모리 무게중심 = 시간측정·통계·기억(노트).** 우리는 의도적으로 **안 가는 길**(1페이지·한 문장·소셜). → 타이머/속도/ETA/통계 코어는 **차용 안 함**.
2. **스트릭이 핵심 리텐션**(~8컷)인데 **비경쟁 톤**("조용히 쌓이는 하루의 독서가 큰 힘이 됩니다", "시작이 가장 멋진 법이에요"). XP·랭킹 없이. → 우리 反-Duolingo 방향과 동일 결, **카피 톤 차용 가치**.
3. **"표지가 쌓인다"** — 서재(완독 표지+별점 그리드) + 독서달력(표지가 날짜 채움). → **표지 책장 전시 + 반별점** 확정.

## 3. 우리 채택 4종 — 캡쳐로 확정/정련

| 우리 피처 | 북모리 실제 구현 | 정련점 |
|---|---|---|
| **사후 감상(노트)** | "책 속 문장(인용)↔페이지(감상)" 유형 토글 + 페이지 메타("몇 페이지의 내용인가요?") + 사후편집 | 우리 `한 문장 + my_note` 구조와 일치. **에디터 레퍼런스** |
| **표지 책장** | 서재=완독 표지+별점 그리드, 달력=표지가 날짜 채움 | "표지가 쌓인다" 비주얼 확정 |
| **반별점** | 서재·달력에 별 5칸(반별 포함) | 0.5 단위 확정 |
| **무작위 회상** | 랜덤 회상 대신 노트별 **"1번 읽었어요" 복습 카운트** | 우리 "랜덤 과거 문장"은 **차별점** — 유지 |

## 4. 베타 분류

- **MUST (불변)**: Supabase·OAuth·UI→DataStore·핵심루프·실소셜 — 북모리 캡쳐와 무관, 그대로.
- **SHOULD (계휴)**: 사후감상(노트 유형·페이지 메타) · 반별점 · 표지책장 · 무작위회상
- **SHOULD (승원)**: 스트릭 **비경쟁 카피 톤**
- **DEFER (베타 후)**: 이미지카드공유(승원) · 독서달력 · 스트릭 복구 · 타이머/통계/속도 · 구매기록 · 백업/머지
- **기각**: OCR 문장캡처 · 바코드(유료/외부 API·네이티브 — 기존 방침)

## 5. 승원 핸드오프 (design/표현 — 계휴가 직접 안 건드림)

- **스트릭 마이크로카피 톤**: 비경쟁·따뜻 ("조용히 쌓이는 하루의 독서가 큰 힘이 됩니다" 류). `design.md`.
- **문장 이미지 카드 공유** 레퍼런스: 인용+출처+워터마크 · 단색/그라데이션/이미지 배경 · 1:1 · 저장/공유. **후순위**.
- **완독 confetti 톤** 참고(우리 세리머니 이미 보유).

## 6. 유의

- **마이크로카피 일관성 반면교사**: 북모리는 같은 복사 동작에 토스트 2개("복사했어요"/"복사되었습니다") 공존 → 우리는 카피 **일원화**.
- 북모리 등록 폼이 과대(7컷) — 우리는 알라딘 자동채움 + **최소 필드**로 마찰 줄이는 방향(반대 전략).
