### 5.5 마을 탭

**마을 = 책 1권 단위 그룹.** 같은 책을 챕터 단위 마일스톤으로 나눠 마감일까지 함께 달린다. 완독 시 자동 아카이브된다.

**마을 상태 정의**:

| 상태 | 설명 | 표시 |
|---|---|---|
| 활성 | 마일스톤 진행 중 | 진행 바 + D-day |
| 완료 | 완독 후 자동 아카이브 | ✅ 완료 |

**CTA 우선순위**: 마을 찾기 = Primary(채운 버튼), 마을 만들기 = Secondary(외곽선). 신규/기존 모두 동일.

**참여 중 마을 카드**: 책 표지 · 마을명 · 파트/전체 · D-day · 진행 바(파트 기준) · 오늘 완료 인원. 완료 상태는 ✅ 뱃지로 대체.

**지난 마을**: 기본 접힘. 나가기·완료 마을 포함. 탭 시 마을 내부(읽기 전용) 진입 가능.

**추천 공개 마을**: 내가 현재 읽는 책의 마을 우선 노출 → 그 외 참여자 수 많은 순. 비가입자에게 마을명·인원만 노출. 없으면 섹션 미노출.

#### 5.5.2 마을 개설 플로우

| 단계 | 입력 |
|---|---|
| 책 선택 | 내서재 또는 검색에서 1권 |
| 마을 정보 | 이름 · 소개(선택) |
| 공개 설정 | 공개(검색 노출) / 비공개(코드·링크 초대) |

**👥 멤버 탭 — 참여자 그리드 (랭킹 통합)**

랭킹 기준: **전체 책 진척률 %** (읽은 페이지 / 책 전체 페이지) → 동점 시 먼저 기록한 시간 순 → 동점 시 스트릭 높은 순.

**불빛 ●/○ 기준**: 오늘 어떤 책이든 기록하면 ● (스트릭 기준과 동일). 마을 진척률은 마을 지정 책 기록만 반영.

**그리드 표시**: 15명 이하 = 전원 3열. 16명 이상 = 상위 15위까지 그리드, 16위부터 한 줄 목록.

```
  1위            2위            3위
┌──────────┐  ┌──────────┐  ┌──────────┐
│  🏰    ●│  │  🏡    ●│  │  🏠    ○│
│  yunji   │  │   kim    │  │   lee    │
│ 진도 75% │  │ 진도 50% │  │ 진도 3%  │
│          │  │          │  │[🪱콕찌르기]│
└──────────┘  └──────────┘  └──────────┘
```

> **진척률 계산**: `읽은 페이지 / 책 전체 페이지 × 100`. 완독 = 100%.

| 카드 요소 | 규칙 |
|---|---|
| 순위 번호 | 카드 외부 상단 |
| 둥지 이모지 | 마을 책 진척 단계 ([nest.md §5.2](./nest.md)) |
| 불빛 (●/○) | 카드 우측 상단 코너. 오늘 어떤 책이든 기록 완료 → 골드 글로우 |
| 진척률 | `진도 XX%` 형식 |
| 콕찌르기 버튼 | **비공개 마을 전용.** 카드 내부 하단. 불 꺼진(○) 멤버에게만. 탭 후 당일 비활성 |
| 카드 탭 | 해당 멤버 프로필로 이동 |

**스트릭 연동**: 마을 체크인 = 개인 스트릭과 동일 기준 ([systems.md §6.1](./systems.md)). 여러 마을 동시 참여 가능.

#### 5.5.5 마을 나가기

"탈퇴" 대신 **나가기**로 통일. 이력은 항상 보존.

| 진입 경로 | 상황 |
|---|---|
| 설정 → [나가기] | 언제든 자의로 마을 떠남 |
| 관리자 강퇴 | 관리자에 의해 나가짐 |

**나가기 동작**:
- 마을 멤버 목록에서 제거
- 지난 마을 목록으로 이동 (이력 보존)
- 남긴 한 문장·게시판 의견 유지
- 확정된 파트 랭킹에서 사라짐

**관리자 나가기 조건**: 권한 이양 완료 후 나가기 가능. 이양 대상이 없으면 마을 삭제만 가능.

#### 5.5.6 마을 설정

헤더 [⚙️] → 설정 화면.

**멤버 공통**:

| 기능 | 설명 |
|---|---|
| 초대 링크 복사 | `rgo.app/v/x7k2p` |
| 초대 코드 복사 | 6자리 대문자 |
| 유저 검색 초대 | 닉네임 검색 → 초대 |
| 알림 설정 | 콕찌르기(비공개만) · 게시판 새 주제 · 파트 마감 D-3/D-1 · 멤버 완독 (ON/OFF) |
| 나가기 | 관리자는 권한 이양 후 가능 |

**공동관리자 전용** (공동관리자에게만 노출):

| 기능 | 설명 |
|---|---|
| 마일스톤 수정 | 파트 챕터 범위 · 마감일 변경 |

**관리자 전용** (관리자에게만 노출):

| 기능 | 설명 |
|---|---|
| 마을 정보 수정 | 이름 · 소개 · 공개/비공개 |
| 정원 수정 | 현재 인원 이하로 줄이기 불가 |
| 공동관리자 지정·해제 | 멤버 선택 → ⭐ 뱃지 |
| 멤버 관리 | 강퇴 · 권한 이양 |
| 마을 삭제 | ⚠️ 확인 다이얼로그 필수 |
