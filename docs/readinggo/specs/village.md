### 5.5 마을 탭

**마을 = 책 1권 단위 그룹.** 같은 책을 같이 읽으며 마감일까지 함께 달린다. 친구 + NPC 둥지를 그리드로 본다.

#### 5.5.1 마을 목록 화면

```
┌─────────────────────────────────────┐
│ 🏘️ 마을                  [+ 마을 만들기]│
├─────────────────────────────────────┤
│ 내가 참여 중인 마을                   │
│ ┌─────────────────────────────────┐ │
│ │《사피엔스》 팀 4주 완독          │ │
│ │ 8명 · 2/4 파트 · D-12           │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ 공개 마을 둘러보기                    │
│ [KAIST 독서 챌린지 · 12명 · D-8 신청]│
└─────────────────────────────────────┘
```

- 참여 중 마을 카드: 책 표지 · 마을명 · 인원 · **현재 파트/전체** · D-day
- 공개 마을 목록: 제목·책·인원·D-day·[신청]. 비가입자에게는 목록·제목·인원만 노출(피드·멤버 현황은 가입자 전용)

#### 5.5.2 마을 개설 플로우

| 단계 | 입력 |
|---|---|
| 책 선택 | 내서재 또는 검색에서 1권 |
| 마을 정보 | 이름 · 소개(선택) |
| 공개 설정 | 공개(검색 노출) / 비공개(코드·링크 초대) |
| **마일스톤** | 책을 **N개 파트로 분할 → 각 파트 마감일** 지정 |
| 멤버 초대 | 링크·코드 공유 또는 유저 검색 |

데이터: `villages` / `village_parts` / `village_members` ([backend.md §7.3](./backend.md)). 생성 = `DataStore.villages.create`.

#### 5.5.3 마을 참여

- **공개**: 도서 상세([profile.md](./profile.md)) → "이 책 읽는 마을" → [참여]
- **비공개**: 초대 코드 입력 / 링크 수신 → 1탭 입장

```
├─────────────────────────────────────┤
│ ┌──────┐┌──────┐┌──────┐            │ ← ② 참여자 둥지 그리드 (3열)
│ │🪱 ●  ││🪱 ○  ││🪱 ●  │            │
│ │ 🏠   ││ 🪹   ││ 🪵   │            │

├─────────────────────────────────────┤
│ ③ 오늘의 한 문장 하이라이트          │
│ ④ 파트별 랭킹   ⑤ 마을 게시판        │
└─────────────────────────────────────┘
```

**① 마일스톤 진행 바**: 현재 파트/전체 파트 · 파트 마감 D-day

**② 참여자 둥지 그리드**:

| 요소 | 규칙 |
|---|---|
| 둥지 아이콘 | 그 사람의 **마을 책 진척 단계** 이모지 ([nest.md §5.2](./nest.md)) |
| 불빛 (●/○) | 오늘 기록 완료 → 골드 글로우 / 미완료 → 회색 |
| 스트릭 미니 | `🔥 N` (Pixel 폰트). 0이면 "쉼 중" |
| 페이지 | `p.N` (마을 책 진도) |
| 액션 | 불빛 OFF: 카드 좌상단 🪱 **콕찌르기** (일 1회) |

**콕찌르기 (🪱)**:
- 불 꺼진 친구 카드 🪱 클릭 → 토스트 `"@{handle}에게 🪱 콕찌르기를 보냈어요!"` → `.sent` (opacity 0.25)
- 일 1회 (`pokes` 테이블 + KST 자정 리셋, [backend.md §7.3](./backend.md)). `DataStore.pokes.send`
- NPC에게는 못 보냄 (자정 후 곧 불빛 ON)
- **현재 정의**: 독려 넛지만 (수령자 게임 효과 없음). 효과 연계는 향후
- **받은 콕 섹션**: "@{handle}가 콕 보냈어요 · 오늘도 같이 읽자🐦" + 시간

**③ 오늘의 한 문장 하이라이트**: 오늘 멤버들이 남긴 한 문장 최신 2~3개. 스포일러 블라인드 적용 ([social.md §5.7.1](./social.md))

**④ 파트별 랭킹**: **마을 지정 책 기록만** 카운트한 누적 페이지 순위. 다른 책 기록 미포함. 완독자 🏆

**⑤ 마을 게시판**: 마을 주민(친구+NPC) 자유 감상 게시글. 한 문장 피드와 차이: 게시판=자유 텍스트, 피드=`sentences` 기반. Phase 0 NPC 시드 3개, Phase 1 입력→저장

**친구 상세 시트** (참여자 카드 탭): 닉네임·스테이지·NPC 여부·bio·읽는 책 칩·오늘의 한 문장. 책 칩 탭 → [profile.md](./profile.md) 책 상세

**스트릭 연동**: 마을 내 체크인 = 개인 스트릭과 동일 기준 (**책 무관 유저 단위**, [systems.md §6.1](./systems.md)). 여러 마을 동시 참여 가능 — 둥지 탭 기록이 마을 진행률에 자동 반영

#### 5.5.5 Phase 배치

| 기능 | Phase |

| 완독 레이스 실시간 | 2 |
