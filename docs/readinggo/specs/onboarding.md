## 4. 유저 시나리오 — 신규 가입 여정

각 단계는 화면 단위이며, **Phase 0 데모에서 모두 클릭으로 확인 가능해야 한다.**

### A. 진입 (비로그인)

- 헤더 슬로건 "하루 한 페이지, 한 문장에서 시작해요", 중앙 CTA `시작하기`
- 슬라이드·튜토리얼 없음
- CTA → **C** (책 등록)

### B. 사전 질문

**삭제.** 1일 1페이지 1문장이 유일한 목표.

### C. 첫 책 등록 (가입 전 가능)

**C-1. 검색 화면**

- 검색창: ISBN / 제목 / 저자
- 클라이언트 fuzzy 매칭 (`books.tsv` 위에서. 한글 자모 분해 `es-hangul` + Fuse.js)
- 결과 카드: 표지 / 제목 / 저자 / 총 페이지
- 검색어 비어있을 때 하단 추천 탭: **요즘 Top 10** / **스테디 Top 10** (Phase 0 하드코딩 큐레이션)
- 결과 없을 시 "직접 등록" → C-2
- 우상단 "📚 내 서재" → 프로필 탭

**C-2. 확인 / 직접 등록 + 오늘의 한 문장 입력**

한 화면에서 책 등록 + 첫 기록 완료.

- 상단: 선택 책 표지·제목·저자 (직접 등록 시 제목·저자·총페이지 폼)
- "현재까지 읽은 페이지" 숫자 입력 (기본 0)
- **구분선**
- "오늘의 한 문장 🐦":
  - "어느 페이지에서?" 숫자 입력 (기본값 = 현재 페이지)
  - 텍스트 영역, placeholder "마음에 든 한 줄을 적어주세요 (최대 200자)"
  - 문자 수 카운터. 1자 이상이어야 CTA 활성
- 자동 임시 저장 (localStorage `rg_pending_sentence`)
- CTA `이 책으로 시작` → 이 책이 **활성 책** 지정 → **D**

### D. 세리머니 (성공 화면)

```
┌─────────────────────────────┐
│   [confetti 18조각, 8색]     │
│      🐦 (참새 bounce)        │
│      훌륭해요!                │
│   첫 한 문장을 남겼어요!       │
│  ┌─────┬─────┬─────┐        │   ← 보상 카드 3그리드
│  │ 🔥  │ ⚡  │ 🪵  │        │
│  │스트릭│+10XP│둥지+ │        │
│  └─────┴─────┴─────┘        │
│  [내일도 짹 →]               │
└─────────────────────────────┘
```

Confetti: 18조각, 8색 `#3FD17F #FFC233 #FF8A3D #5AB5F0 #F08A9A #B690F0 #2EB867 #FFD66B`, `duration 1.6~3s`, `delay 0~0.25s`, ~3.2초 후 정리, `z-index 110`.

- 가입 전이면 CTA `Google로 계속` (OAuth 직행), 로그인 상태면 홈 복귀

### E. 가입

- 별도 랜딩 없음. D 세리머니 CTA가 OAuth 트리거
- OAuth 성공 후: localStorage 임시 데이터 → Supabase 동기화 ([backend.md §7.7](./backend.md)), 닉네임 미입력 시 E-1, 바로 **H**(홈)

**E-1. 닉네임 설정**

- 입력 `@nickname`, 규칙 표시 (영소문자/숫자/한글/언더스코어, 2~16자, 중복 불가, 금칙어)
- 디바운스 300ms 후 `/rpc/check_handle`

### G. 친구 / NPC

가입 직후 자동: NPC 2명 자동 팔로우 `@book_bear`(책읽는곰돌이) / `@activist_raccoon`(활자라쿤). NPC는 UI상 일반 유저와 동일, 마을에서 ` · NPC` suffix.

**NPC 풀 (Phase 0 검색 가능 4명)**:

| 핸들 | 이름 | 스테이지 | 성격 |
|---|---|---|---|
| `@book_bear` | 책읽는곰돌이 | 3 | 따뜻한 감상 |
| `@activist_raccoon` | 활자라쿤 | 4 | 분석적, 인용 |

**상호작용 (v7)**:
- **소셜 피드 반응 = 짹 1버튼** (한 문장 좋아요, 토글). 받으면 XP +1 ([systems.md §6.3](./systems.md))
- **마을 둥지 카드 반응 = 콕찌르기(🪱) 1버튼** (불 꺼진 친구에게만, 일 1회) — [village.md](./village.md)
- 짹 = 작성된 한 문장에 대한 좋아요 / 콕찌르기 = 오늘 안 읽은 친구에게 독려 넛지

NPC 운영: pg_cron(또는 Phase 0 시드) 진도 증가 + 시드 한 문장 게시. LLM 호출 없음. [systems.md §6.4](./systems.md).

### H. 홈 도착 (Day 1)

기본 탭 = **둥지**.

```
┌─────────────────────────────────────┐
│ 🐦 readingGo  🔥1 ⚡+10 Lv1 🏰0  📚  │ ← 상단 바 (성 개수 포함)
├─────────────────────────────────────┤
│ 🪵 나뭇가지 자리      20%            │ ← 둥지 진화 배너
│ ▓▓░░░░░░  사피엔스 · 5/300p          │   (표지 탭 → 활성 책 전환)
├─────────────────────────────────────┤
│        🐦                            │ ← 둥지 비주얼 (진척률에 따라 자람)
│      (참새가 둥지에)                  │
│      [오늘 기록하기]                  │
└─────────────────────────────────────┘
│ [ 둥지 · 마을 · 소셜 · 프로필 ]       │ ← 하단 탭
```

하단 탭(4): **둥지 / 마을 / 소셜 / 프로필**
- **둥지**: 활성 책의 둥지 자람 + 일일 미션 ([nest.md](./nest.md))
- **마을**: 책 단위 그룹 + 파트 마일스톤 ([village.md](./village.md))
- **소셜**: 신규 시작러 + 전체 공개 한 문장 피드(짹) ([social.md](./social.md))
- **프로필**: 내 서재 · 성 컬렉션 · Export · 설정 ([profile.md](./profile.md))
