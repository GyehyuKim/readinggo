# 개인정보처리방침 (Privacy Policy) — 정본 + 게시 설계

> **신설 (#757, 2026-06-17)**: ReadingGo는 이메일(Google OAuth)·독서기록·행동데이터를 수집하므로 PIPA상 처리방침 게시가 의무. 동의 모델은 [analytics.md §5](./analytics.md), 측정 이벤트는 §3.1.
> **편집 정책**: 처리방침 본문 변경은 이 파일 PR로. spec-only 준수. 코드(링크·화면 배선)는 후속.
> **주의**: 아래는 운영용 정본 초안이다. 정식 런칭·대규모면 법무 검토 권장(작성자는 변호사 아님). 시행일·수탁사 리전은 게시 전 확정.

## 1. 수집하는 개인정보 항목

| 구분 | 항목 |
|---|---|
| 계정(로그인 시) | 이메일, 표시이름·닉네임, 프로필 이미지(Google 제공) |
| 서비스 이용 | 독서 기록(책·쪽수·세션·스트릭·XP), 한 문장·감상, 짹(좋아요), 둥지/완독 |
| 문의 | 문의 내용, 회신용 이메일, 앱 버전 |
| 행동 데이터(분석) | 페이지뷰·클릭·이벤트(식별자 없는 익명) / **세션 리플레이**(선택 동의 시) |
| 자동 생성 | 쿠키·기기/브라우저 정보, 접속 로그 |

- 게스트(비로그인)는 계정정보 없이 **로컬(localStorage)**로만 동작. 로그인 시 Supabase로 적재.

## 2. 수집·이용 목적

서비스 제공(독서 기록·소셜·둥지), 본인 식별·인증, 문의 응대, **서비스 개선·통계 분석**(익명), 선택 동의 시 세션 리플레이로 사용성 개선·LLM 독서 파트너 품질 향상.

## 3. 보유·이용 기간

회원 탈퇴 또는 동의 철회 시까지. 단 법령상 보존 의무가 있으면 해당 기간. 행동 분석 데이터는 PostHog 보존 정책에 따름. 미동의 항목(리플레이·식별·LLM)은 수집하지 않음.

## 4. 제3자 처리위탁 및 국외 이전

서비스 운영을 위해 아래에 처리를 위탁하며, 일부는 **국외**로 이전된다(PIPA 고지).

| 수탁사 | 처리 항목 | 목적 | 이전 국가 |
|---|---|---|---|
| Supabase | 계정·독서기록·문의 | DB·인증 호스팅 | 국내(대한민국, AWS 서울 `ap-northeast-2` — 2026-07-02 Management API 실측, #1126) |
| PostHog | 행동 이벤트 / 리플레이(선택) | 분석·세션 리플레이 | 미국 |
| Google | 이메일·프로필 | OAuth 로그인 | 미국 |
| Upstage | 책 사진·한 문장 텍스트 | OCR·LLM 독서 파트너 | 국내(한국) |

- 이전 일시·방법: 서비스 이용 시점에 네트워크 전송. 외부 판매·광고 목적 제공 없음.

## 5. 이용자의 권리

열람·정정·삭제·처리정지·동의철회를 언제든 요구할 수 있다. 행사 방법:

- 앱 **서재(내 서재) → 데이터 내보내기**(JSON)로 열람(#568 이후 내보내기는 설정→서재로 이동 · 드리프트 정정 2026-07-09), **설정 → 동의 토글**로 선택 항목 철회(리플레이·식별·LLM 즉시 중단).
- 계정·데이터 삭제, 기타 요청: 아래 연락처.

## 6. 쿠키

서비스는 분석을 위한 **first-party 쿠키**(PostHog distinct_id)를 사용한다. 브라우저 설정으로 차단 가능하나 일부 통계가 수집되지 않을 수 있다. 별도 광고·추적 제3자 쿠키는 사용하지 않는다.

## 7. 개인정보 보호책임자 / 연락처

문의·권리 행사: **`readinggo.admin@gmail.com`** (앱 설정 → 운영자에게 문의).
