## 1. 한 줄 정의

> "3주 전 이 문장에 이렇게 답했는데 — 지금은 어때요?"

과거에 남긴 문장과 내 답변을 재소환해, 생각의 변화를 스스로 발견하게 한다.
이게 ReadingGo만의 edge: 수개월 이상 써야 생기는 맥락 — ChatGPT·북모리가 줄 수 없는 것.

---

## 2. 트리거 조건

### 2.1 발동 시점
- 앱 진입(둥지 로드) 시 백그라운드 체크
- **발동 조건**: 아래 중 하나 이상 충족한 문장·대화가 있을 때
  - 대화(Q/A)가 저장된 문장 중 마지막 대화로부터 **14일 이상** 경과
  - 같은 책의 다른 문장과 대화한 이후 **7일 이상** 경과 (책 안에서 생각 연결)
- **최대 노출 빈도**: 1일 1회 (매 진입마다 뜨지 않음)
- **조건**: 대화(Q/A)가 1개 이상 저장된 문장만 대상 (답변 없는 문장 제외)

### 2.2 선택 알고리즘 (우선순위 순)
1. 가장 긴 내 답변이 달린 문장 (생각이 많이 담긴 것)
2. 14일 이상 재소환 안 된 문장
3. 랜덤 (동률 시)

## 3. UI

### 3.1 둥지 홈 카드 (비침습적)

```
┌─────────────────────────────┐
│  💬 참새                     │
│                             │
│  23일 전, 이 문장을 남겼어요  │
│  ─────────────────────────  │
│  " 완전한 행복이란 존재하지   │
│    않지만, 완전한 절망도      │
│    존재하지 않는다 "          │
│  ─────────────────────────  │
│  내가 썼던 것:               │
│  "지금 내 상황이랑 너무       │
│   비슷해서 울뻔했어"          │
│                             │
│  지금은 어때요?              │
│                             │
│  [ 다시 대화하기 ]  [ 나중에 ]│
└─────────────────────────────┘
```

- 둥지 메인 카드 영역에 **조용히** 삽입 — 전체 화면 팝업 아님
- "나중에" 탭 시 오늘 하루 숨김 (내일 다시 체크)
- 스크롤로 지나칠 수 있음 (강요 없음)

### 3.2 대화 재개 모달

"다시 대화하기" 탭 시 CompanionModal 진입:
- 상단: 원래 문장 + 책 제목
- 중단: 이전 대화 스레드 (읽기 전용, 흐릿하게)
- 하단: 참새 "지금은 어때요?" 질문 + 입력창
- 답하면 대화 스레드에 추가 저장
