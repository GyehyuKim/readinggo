## 핵심 발견

### 1. 뿌리 핵심 가치

> **"흔들리지 않는 자기 중심 (anchored self)"**
>
> 많이 읽는 게 목적이 아니라, 읽음으로써 쌓인 것들이 나의 중심축이 되는 것.

- 독서의 보상 = 순간 감정이 아니라 **정체성 형성**
- 목표 = "어떤 사람이 되는가" — 박학다식하고 현명함
- 시간축 = 즉각적이 아닌 **점진적·장기적 축적**

### 2. 사용자도 같은 뿌리

"ReadingGo 사용자도 같은 뿌리를 갖고 있다" — 세션 중 확인.
→ 이 뿌리는 개인 동기가 아니라 **제품의 존재 이유**.

### 3. 계휴 Y-tree와의 비교 (2026-06-08 세션)

| | 계휴 | 윤지 |
|---|---|---|
| **뿌리** | "나를 비추는 거울" + 공명 (타인과 연결) | "내 생각의 중심" (정체성 형성) |
| **시간축** | 순간 (고양감·공명) | 장기 (축적→중심) |
| **소셜** | 뿌리에 있음 (공명) | 뿌리에 없음 (완전히 내면) |
| **수렴점** | 둘 다 "한 문장이 쌓인다"는 축적 감각 | |

**차이에서 나온 질문:**
- "고양감(uplift)"은 윤지 트리의 왜 #2(감정의 폭) 수준 — 뿌리까지 내려가면 "중심·정체성"이 나온다.
- 피치 중심어로 "고양감" vs "내 생각의 토대" 중 어느 쪽이 더 ReadingGo다운가?
- "1인 독서모임" 프레임은 계휴 트리엔 자연스럽지만 윤지 트리엔 약간 어긋난다 (소셜 뉘앙스 vs 내면 성장).
