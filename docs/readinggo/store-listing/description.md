# Play 스토어 리스팅 카피

> Refs #1128, GTM §3.1 (#1124). Console 입력용 정본 — 수정 시 이 파일 PR 먼저.

## 앱 이름 (30자 제한)

```
리딩고 — 하루 한 문장 독서 습관
```

## 짧은 설명 (80자 제한)

```
책을 더 쉽게, 더 깊게. 하루 한 쪽, 한 문장으로 독서 습관을 만들어요.
```

## 전체 설명 (4000자 제한)

```
사놓고 3페이지에서 멈춘 책, 다들 한 권쯤 있잖아요.

의지가 없어서가 아니에요. 하루의 호흡에 책 읽을 자리가 없었을 뿐.
리딩고는 그 자리를 만들어요 — "오늘 30분 독서" 같은 큰 약속 대신,
오늘 한 쪽, 한 문장. 그것만 지키면 오늘은 성공입니다.

■ 하루 한 문장이 만드는 것

읽다가 마음에 든 한 줄을 옮겨 적으면 오늘의 기록이 끝나요.
페이지 숫자만 채우는 게 아니라, 내가 직접 고른 문장이 남으니까
"진짜 읽었다"는 증거가 쌓입니다.

■ 매일 이어가게 하는 장치들

· 스트릭 — 하루도 빠짐없이 이어가는 연속 기록
· 둥지 — 기록할수록 자라는 나만의 참새 둥지 (5단계 진화)
· 세리머니 — 오늘의 기록을 마치면 작은 축하가 기다려요

■ 문장이 쌓이면 자산이 됩니다

책마다 내가 고른 문장들이 컬렉션으로 정리돼요.
언제든 Markdown으로 내보낼 수 있어요 — 리딩고를 떠나도
당신의 문장은 당신 것입니다.

■ 같은 책을 읽는 사람들

지금 같은 책을 읽는 다른 사람이 남긴 한 문장을 만나보세요.
같은 책에서 서로 다른 문장을 고른 걸 보는 재미가 있어요.
친구 요청도, 수락도 없어요. 부담 없는 단방향 팔로우.

■ 기록의 마찰을 줄였어요

· 책 사진 한 장 찍으면 문장을 자동으로 추출 (OCR)
· 바코드 스캔으로 책 등록
· 로그인 없이 게스트로 먼저 둘러보기

--

하루 한 쪽이면 충분해요. 오늘부터 시작해 보세요.

문의: readinggo.admin@gmail.com
개인정보처리방침: https://readinggo.hyuniverse.workers.dev/privacy
```
