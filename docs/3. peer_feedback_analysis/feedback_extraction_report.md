## 🛠️ 추출 프로세스 및 분석 기술 개요

본 아카이브는 HTML 파일 내에 내장된 데이터 소스를 안전하게 추출하기 위해 아래 기술을 사용했습니다.
1. **JSONDecoder.raw_decode 기반 파싱**: HTML 파일 내부 자바스크립트 블록 내의 거대한 중괄호(`{ ... }`)쌍을 정확하게 추적하여, 문자열 내부에 포함될 수 있는 세미콜론이나 기호에 구애받지 않고 원본 데이터를 완벽하게 JSON 객체화하였습니다.
2. **BOM UTF-8 인코딩 적용**: 윈도우 환경 및 한국어 Excel에서 CSV 파일을 바로 더블 클릭해 열어도 인코딩이 깨지지 않고 정상적으로 한글이 표시되도록 `utf-8-sig` 형식의 인코딩을 적용했습니다.
3. **결측치 정제**: 피어 리뷰 데이터 중 누락값(예: `nan`, `-`, `none` 등)이 포함된 경우를 제외하고 유효한 피드백만을 스프레드시트에 기입하여 분석의 가치를 높였습니다.
